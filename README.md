# 自建图书馆

文件夹与分类：

| 目录             |                                      |
| ---------------- | ------------------------------------ |
| boundless-sea    | 无法归类的书籍或者没有时间归类的书籍 |
| comic            | 漫画                                 |
| history&politics | 历史与政治文献和书籍                 |
| philosophy       | 哲学思想书籍                         |
| SF&fantasy       | 科幻与奇幻作品，包含轻小说           |
| spells&skills    | 职业专业技能和神秘学书籍             |
